# Couple App

A mobile app for couples to share moments, chat, and mark special dates. Built with Expo and React Native.

## Features
- Red/Blue Theme Selection
- Partner Invitation System
- Chat
- Photo Feed
- Special Dates Calendar

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the app:
   ```bash
   npx expo start
   ```

3. Scan the QR code with Expo Go (Android/iOS) or press `w` for Web.
